## Getting Started

Run before:

```bash
git clone git@github.com:rafaelandrade/save4me.git
yarn
```

### Run tests

```bash
yarn workspace backend test
```

### Run in browser

```bash
yarn workspace backend start
```

And open [http://localhost:3000](http://localhost:3000)
